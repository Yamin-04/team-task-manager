# Team Task Manager 🚀

A full-stack MERN web application for managing team tasks and tracking workflow progress.

---

## 📌 Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Task Status Tracking
  - Pending
  - In Progress
  - Completed
- REST API Integration
- MongoDB Atlas Database
- Live Deployment

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Vercel (Frontend)
- Render (Backend)

## ⚙️ Installation

### Clone Repository

```bash
https://github.com/Yamin-04/team-task-manager
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
node server.js
```
